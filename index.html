Qual a diferença entre massa corrida PVA e acrílica? +

A massa corrida PVA é indicada exclusivamente para uso interno, pois não resiste à umidade e à chuva. A massa corrida acrílica pode ser usada tanto em ambientes internos quanto externos (fachadas, áreas descobertas), pois tem maior resistência à água e às variações de temperatura. Para paredes externas, sempre escolha a versão acrílica. Em caso de dúvida, nossa equipe orienta pelo WhatsApp.

Como aplicar textura grafiato em parede interna? +

Para aplicar grafiato em parede interna com resultado profissional:

  1. Limpe a superfície removendo poeira, gordura e partes soltas.
  2. Aplique a seladora (já inclusa no nosso grafiato) e aguarde secar (4h).
  3. Aplique o grafiato com rolo de pelo longo ou projetor de textura.
  4. Faça movimentos uniformes para criar o efeito rústico desejado.
  5. Aguarde 24h antes de uma segunda demão se necessário.
O grafiato Qualy Quimy funciona em fachadas externas? +

Sim! O grafiato Qualy Quimy tem base acrílica de alta resistência, indicado tanto para ambientes internos quanto externos, incluindo fachadas. A versão com seladora inclusa é especialmente prática para fachadas, pois elimina uma etapa de preparo. Para áreas com exposição direta à chuva, aguarde 24h entre demãos.
